Selamat Datang

Terimakasih telah membeli script website kami. Kami berharap anda menjadi salah satu pelopor untuk membeli script pada orang yang tepat dan resmi, sebagai bentuk apresiasi terhadap pengembang website.

Upload File

  • Login Ke hosting cpanel anda!
  • Buka Main Root dari domain kamu. jika tidak mengunakan banyak domain, sialahkan letakkan pada folder public_html
  • Pada menu File Manager Folder public_html Klik Upload.
  • Pilih File Anda, dan klik Upload
  • Setelah selesai upload, silahkan kembali ke main root domain anda
  • Cari file dan extract file

    • Konfigurasi Database

  • Pastikan anda telah membuat database, jika belum silahkan lihat tutorial
  • Rubah Code di folder lib>config.php
    •               
                    //Config Database
                    date_default_timezone_set('Asia/Jakarta');
                    $db_server = "localhost";
                    $db_user = ""; //user database
                    $db_password = "";//password database
                    $db_name = ""; //nama database
                    $cfg_baseurl = "https://namadomain.com"; //nama domain tanpa akhiran /
  • Jangan lupa untuk simpan

    • Konfigurasi Tripay

    Berikut untuk konfiguasi pembayaran otomatis tripay

  • Hal pertama, sudah mendaftar akun ditripay
  • 1. Klik merchant pada sidebar
  • 2.Klik tambah merchant
  • 3.Isi data lengkap

    • Sesuaikan dengan data berikut:

  • callback : namadomainkamu.com/tripay/callback.php
  • Whitelist IP gunakan share ip hosting kamu. jika menggunakan domainesia, silahkan input IPv4 dan IPv6 dari dashboard client layanan hosting. Berlaku untuk semua hosting yang menggunakan ip keduanya. jika hanya IPv4 saja, cukup tambahkan 1 saja

  • Buka folder lib > mainconfig.php dan ganti sesuai dengan merchant kamu
    •               
                // TRIPAY
                $tripay_merchant_code = "Kode_merchant";
                $tripay_private_key = "Private_key";
                $tripay_api_key = "API_KEY";
  • Jangan lupa simpan

    • Pengaturan Cron Update status dan refund

    Untuk mengatur supaya pesanan kamu dan deposit terupdate, silahkan setting cron website kamu.

  • Login cpanel, dan cari cron
  • Setelah itu buat new cron atau new task
  • centang menjadi **** dan comand tasknya:
    •               
                * * * * * wget -O - https://yoursite.com/cron/status.php >/dev/null 2>&1
              
            
                  
                * * * * * wget -O - https://yoursite.com/cron/refund.php >/dev/null 2>&1

    Tambah API Provider

    Silahkan login ke akun admin, dan masuk halaman admin. Pilih menu Api Layanan pada sidebar, dan klik tambah sesuai kebutuhan anda!

    Update Layanan

    Silahkan login ke akun admin, dan masuk halaman admin. Pilih menu Layanan Sosmed>Update Sosmed pada sidebar!
