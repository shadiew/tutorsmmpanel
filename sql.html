Panduan Import Database

Silahkan login ke cpanel terlebih dahalu, jika sudah, silahkan ikuti tutorial berikut

1.Pada menu cpanel, cari database. klik phpmyadmin

2.setelah berhasil masuk kehalaman phpmyadmin, silahkan pilih salah satu nama databasenya

3.Setelah dipilih nama databasenya, silahkan klik import

4.Klik chose file, dan cari databasenya dengan format .sql

5.Setelah dipilih databasenya, silahkan scrool kebawah dan klik import
